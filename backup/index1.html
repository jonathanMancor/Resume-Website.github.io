O.K.I

Welcome to Oki.exe, a web-based role playing game made using HTML5, CSS and Jquery.

Compatability

Oki.exe is meant to be played in fullscreen mode (F11).when oki is loaded, the Website will automatically activate fullscreen mode.

For the best performace, it is recommended to play on a laptop or desktop and using the latest version of Google Chrome as some features are not supported on other browers. Full support for other browers is coming soon.
